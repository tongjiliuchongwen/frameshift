# PHYSMASTER — 自主 AI 物理学家（论文 gap）

PHYSMASTER 让一个 agent 端到端跑完一个被给定/被选定的物理问题，并用一个 LLM critic 判断结果好不好、新不新，把原本需要资深博士 1-3 个月的工程性环节压缩到 6 小时内。

已知弱点：LLM critic 会残留幻觉，在评估新概念时可能引入偏见、或漏掉本来可行的探索路径；被否决/走不通的路径要不要留下来不清楚；整套方法依赖高质量文献检索，在新兴或小众子领域文献质量参差会拖垮表现；人类物理学家在回路里到底扮演什么角色也不明确。

> v0.5 三段式示例输入。完整产物见 `outputs/`（系统图 → 杠杆点 →【Gate1】→ 横向发散 →【Gate2】→ 垂直审计 →【Gate3】→ idea 卡），validator exit 0。
